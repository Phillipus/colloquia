Adding a new Resource

If you know its URL

To add a Resource to your Resource Folder, click on it once to highlight the folder. Select New Resource from the Insert Menu, to give you the dialogue box for name entry.

Enter a name and click OK (this need only be a short name if you so wish). This name will appear inside your Resource Folder, and you will be able to add further details in the Main Frame.

Notice that in the tree, the Resource has been added simply by its name that you entered, but in the Main Frame, the full title has been added, as well as various other details. Double click inside each field, enter the information and then press Return to add further details.

If you have put in a URL for the Resource, then you can view this using the built in browser in the bottom half of the Main Frame. Simply click on the URL Button , and the web page will be displayed - but make sure you are connected to the internet when you do this. You can also save and then view a local version, as described in the Saving A Web Resource section.

You can add any Resource which can be downloaded from a web or ftp server, even if it cannot be displayed by the browser - these will simply be saved instead of displayed (see the Saving A Non-Web Resource section).

Directly From the Browser

A typical way of adding Resources is to browse or search the web using the browser, and reference it when you have found one you want to add to your Resource Folder. Simply click on the Create Button in the browser to do this. This will give you a dialogue box and from here, the process is identical to that previously described in the If You Know Its URL Section.

Updating a URL

If you find that a URL has changed for one of your Resources, you can use the browser to find the new location, and then click on Update URL in the browser. This will update the URL in the Resource's URL field (assuming this information is currently displayed - if you do not take care you could easily update the wrong Resource!)
